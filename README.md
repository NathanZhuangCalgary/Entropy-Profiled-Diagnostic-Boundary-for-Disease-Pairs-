# Entropy-Profiled-Diagnostic-Boundary-for-Disease-Pairs-
Entropy-Profiled Diagnostic Boundary for Disease Pairs Test Used To Quantify Uncertainty Across Disease Pairs. 

Abstract:
Bioinformatics increasingly requires analytical approaches capable of capturing uncertainty and complex relationships within high-dimensional biomedical data. Many commonly used statistical methods rely on extensions of existing frameworks, such as post-hoc analyses or confidence interval estimation, which may not fully characterize uncertainty across patient cohorts. To address this limitation, we introduce the Entropy-Profiled Diagnostic Boundary for Disease Pairs, a method that evaluates cohort separation through entropy-based uncertainty profiles. The approach embeds two cohorts within a virtual feature space and evaluates classification certainty along a centroid trajectory using Shannon’s entropy index. For each patient position along this trajectory, the method measures uncertainty relative to a defined number of nearest neighbors (k), producing a continuous entropy profile across the feature space. This framework provides an alternative perspective to traditional variance-based measures, classifiers, or area-under-curve (AUC) analyses by directly quantifying uncertainty in cohort classification. The method generates several diagnostic indices, including maximum and minimum entropy values, transition width and sharpness indices describing boundary irregularities, normalized transition metrics enabling cross-dataset comparison, and identification of patients contributing most strongly to entropy variation. The method was evaluated across twenty synthetically generated datasets spanning five categories: Gaussian separation, logistic probability structures, correlated features, boundary-injected datapoints, and null distributions. Each category included both titer and binary datasets with varying distribution scales. The approach was also applied to a clinical autoimmune dataset containing both titer and binary measurements. Across synthetic datasets, titer-based datasets produced clear entropy transitions across the feature space, and in boundary-injected datasets the method successfully identified all injected datapoints. Application to the clinical dataset revealed patterns consistent with previously reported clinical observations. This work contributes a novel entropy-based framework for analyzing disease cohort boundaries that emphasizes uncertainty profiles and patient-level contributions to classification ambiguity. Rather than functioning as a confirmatory statistical test of association, the method serves as an exploratory analytical tool that characterizes gradients of disease separation while highlighting data points that are often treated as outliers in conventional statistical analyses.
